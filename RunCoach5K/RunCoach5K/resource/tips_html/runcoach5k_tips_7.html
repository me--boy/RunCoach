Function:

Change speed with plan, improve running technology, and strength general endurance.

Practice methods:

  • Run slowly for 20 meters (yards) and then fast for 40 meters (yards).
  • Run fast for 50 meters (yards) and then slowly for 50 meters (yards).
  • Run fast for 100 meters (yards) and then slowly for 60 meters (yards).
  • Speed up suddenly and maintain for a while, then change to steady running or slow running.
  • Run fast and then slow from long distance to short distance (or from short to long) for 400 meters (yards), 300 meters (yards), 200 meters (yards), 100 meters (yards), 80 meters (yards), 60 meters (yards), 40 meters (yards), and 20 meters (yards). Distance of fast running should be short and slow running should be long for beginners. Fast running should be enlarge according to the level improved.
