Function:

Finish running of specified distance with high intensity, then run again after limited time rest, interval run will strength the function of cardiovascular system and anaerobic metabolism capacity.

Practice methods:

  • Interval running for 200 ~ 400 meters (yards), speed should be close or faster than race speed.
  • Interval running for 400 ~ 600 meters (yards), speed should be slower than race speed.

    • Note:

    If speed of interval running is faster than race, rest for a longer time, do the next practice when heart rate is close in 120 times/minutes. If speed of interval running is slower than race, rest for a short time, do the next practice before close in 120 times/minutes.
